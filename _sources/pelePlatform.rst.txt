**************
PelePlatform
**************

PelePlatform (https://github.com/NostrumBioDiscovery/pele_platform) is a Python package, developed at nostrumBioDiscovery, aimed at automatically launching all PELE features with a general, user-friendly command.

For more documentation on the installation and the usage of PelePlatform please refer to https://nostrumbiodiscovery.github.io/pele_platform/
